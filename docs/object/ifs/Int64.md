## 构造函数
        
### Int64
**Int64 构造函数**

```JavaScript
new Int64(Long num = 0);
```

调用参数:
* num: Long, 初始化的值

--------------------------
**Int64 构造函数**

```JavaScript
new Int64(Long hi,
    Long lo);
```

调用参数:
* hi: Long, 高32位数
* lo: Long, 低32位数

--------------------------
**Int64 构造函数**

```JavaScript
new Int64(Int64 num);
```

调用参数:
* num: Int64, 初始化的值

--------------------------
**Int64 构造函数**

```JavaScript
new Int64(String num,
    Integer base = 10);
```

调用参数:
* num: String, 数字字符串
* base: Integer, 数字字符串的编码方式，可以接受 2-16, 32, 64，默认为 10，为 10 时自动识别 0x 编码

### compare
**将 Int64 内的数值与给定数值比较大小**

```JavaScript
Integer Int64.compare(Int64 num);
```

调用参数:
* num: Int64, 指定比较的数值

返回结果:
* Integer, 返回 1 表示 Int64 内的数值比指定数值更大，0 表示相等，-1 表示更小

--------------------------
