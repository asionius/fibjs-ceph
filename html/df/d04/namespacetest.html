§ run()

static Integer test.run ( Integer  loglevel = console.ERROR)
static

开始执行定义的测试模块

参数
loglevel指定进行测试时的日志输出级别，ERROR 时，项目报错信息集中在报告后显示，低于 ERROR 时，输出信息随时显示，高于 ERROR 时，只显示报告
返回
返回测试用例统计结果，正确则返回 0，错误则返回错误个数
